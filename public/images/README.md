# Ảnh Landing Page – Robotimize

Đặt các ảnh sau vào thư mục `public/images/`:

| Tên file              | Mô tả                          | Tỷ lệ khuyến nghị |
|-----------------------|--------------------------------|-------------------|
| `image_hero.jpg`      | Ảnh hero - thiết bị phục hồi  | 4:3 hoặc 16:9     |
| `image_features.png`  | Ảnh tính năng / sản phẩm       | 16:9              |
| `image_testimonial.jpg` | Ảnh đại diện bệnh nhân      | 1:1 (vuông)       |
| `image_cta.png`       | Ảnh nền CTA                    | 16:9 hoặc rộng hơn|

Tất cả ảnh nên có độ phân giải tối thiểu 1200×800px để hiển thị rõ nét trên màn hình Retina.
